🛒 Amazon Frontend Clone (HTML & CSS)

This project is a frontend clone of the Amazon website built using only HTML and CSS. It replicates the layout, design, and basic user interface of Amazon’s homepage to practice web development fundamentals.

📌 Features
Amazon-style homepage layout
Product grid section
Hero banner section
Footer similar to Amazon
Clean and structured UI using pure CSS

🛠️ Technologies Used
HTML5 – Structure of the website
CSS3 – Styling and layout
